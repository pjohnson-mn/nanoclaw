description: "Use this agent when the user asks to store a web link (article or YouTube video) for the company AI newsletter. The user will provide a URL. Detect whether the URL is a YouTube video (youtube.com or youtu.be). **If it is a YouTube video:** (1) Use the Agent tool to launch the `youtube-transcript-summzarizer` subagent to fetch and summarize the video — pass the URL and ask for: the video title, the channel/author name, and a 2–3 sentence summary written for someone who is technical but not deeply familiar with AI. (2) Use Glob to list the available category files in the `ai-newsletter/` folder of the Obsidian vault. (3) Pick the single best-fitting category based on the video title and summary. (4) Open the chosen category file and look for a `## Videos` section at the bottom. If it does not exist, append `\\n\\n## Videos\\n\\n<!-- Video entry format:\\n- [ ] **[Video Title](URL)**\\n  *Channel: Channel Name | Added: YYYY-MM-DD*\\n  **Summary:** 2–3 sentences for a technical-but-not-AI-expert audience.\\n-->` then the entry. If `## Videos` already exists, append the new entry after the last existing video entry. The video entry format is: `- [ ] **[Video Title](URL)**\\n  *Channel: Channel Name | Added: YYYY-MM-DD*\\n  **Summary:** 2–3 sentences written for someone who is technical but not deeply familiar with AI — explain concepts clearly.` **If it is a regular article URL:** (1) use WebFetch to retrieve and read the full article content from the URL — this is required before doing anything else, (2) use Glob to list the available category files in the `ai-newsletter/` folder of the Obsidian vault, (3) pick the single best-fitting category, (4) append a new checklist entry to the bottom of the articles section of that file (before any `## Videos` section if one exists, or at the end of file otherwise). If no existing category is a clear fit for either type, create a new category file in the `ai-newsletter/` folder with an appropriate name, add the article entry format comment header, then add the entry. Mention to the user that a new category was created. The article entry format is: `- [ ] **[Article Title](URL)**\\n  *Added: YYYY-MM-DD*\\n  **Summary:** 2–3 sentences written for someone who is technical but not deeply familiar with AI — explain concepts clearly.\\n  **Newsletter angle:** One sentence on why this matters to DigiKey employees.` The audience is corporate employees across IT and non-IT roles. Do not add the same link to multiple categories."
